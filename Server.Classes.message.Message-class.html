Class Message

source code

" A message object to be passed between threads via Queue Sample Usage: Making a new message that will send terminate from main to display: newMessage = Message("Main", "Display", "Terminate", \{\}) Making a slide that will update Display with a new slide: newMessage = Message("Main", "Display", "Update", \{\}) newMessage.add_content("slide1", "http://google.com")

Attention: It is good practice to use dict accessors as opposed to object attributes
